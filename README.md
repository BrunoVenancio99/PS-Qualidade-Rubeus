Web Quality Test Suite

Suíte de testes automatizados desenvolvida em Python para validação técnica e SEO básico de páginas web.

🚀 Objetivo

Automatizar a verificação de qualidade de páginas web analisando:

✅ Status HTTP (200)

⏱️ Tempo de resposta

🏷️ Existência da tag <title>

🔎 Presença de meta description

📊 Geração automática de relatório HTML

Projeto desenvolvido como prática de automação de testes e validação de qualidade de software.

🛠️ Tecnologias Utilizadas

Python 3.14

Requests

BeautifulSoup4

Pytest

Pytest-HTML
